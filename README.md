# LearnMate-AI-Learning-Coach

## Project Overview
LearnMate is an Agentic AI-powered learning coach that generates personalized learning roadmaps based on a student's interests, skill level, study time, and career goals.

## Technologies Used
- Langflow
- IBM watsonx.ai
- IBM Granite Models
- IBM Cloud

## Features
- Personalized 6-week roadmap generation
- Goal-based learning plans
- Career guidance
- AI-powered recommendations
